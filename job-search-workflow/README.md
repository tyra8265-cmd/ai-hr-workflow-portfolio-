# Job Search Workflow

## Overview

This project outlines a structured AI-supported workflow for organizing a job search.

The goal is to help job seekers review roles, track applications, prepare materials, and follow up with more consistency.

## Problem

Job searches can become scattered quickly.

Roles are saved in different places, applications are hard to track, and follow-up is easy to miss.

Many job seekers need a system that helps them stay organized without applying blindly or sending low-quality materials.

## What the Workflow Can Help With

- Collect job postings
- Score role fit
- Track application status
- Draft value snapshots
- Draft recruiter messages
- Prepare interview notes
- Organize follow-up reminders
- Summarize job search activity

## Example Workflow Stages

1. Job found
2. Role reviewed
3. Fit scored
4. Resume snapshot drafted
5. Application reviewed
6. Application submitted
7. Follow-up scheduled
8. Interview prep created

## Sample Job Review Fields

- Company
- Role title
- Location
- Remote status
- Salary range
- Match score
- Why it fits
- Risks or gaps
- Application status
- Follow-up date

## Human Review Rules

A human should always review before:

- Applying to a role
- Sending a recruiter message
- Updating a resume
- Using AI-generated career materials
- Claiming any experience or metric

## Safety Position

The workflow should never invent experience, exaggerate qualifications, or apply without human review.

AI can help organize the job search.

The human owns the final decision.
