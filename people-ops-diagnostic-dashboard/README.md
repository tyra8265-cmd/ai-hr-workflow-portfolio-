# People Ops Diagnostic Dashboard

## Overview

This project outlines a People Ops diagnostic dashboard for identifying patterns behind recurring HR and workforce problems.

The goal is to help HR teams, consultants, and small businesses see people risks earlier.

## Problem

People problems often show up as isolated issues.

But underneath, there is usually a pattern.

Examples:

- Turnover
- Burnout
- Manager issues
- Broken trust
- Hiring delays
- Poor onboarding
- Low accountability
- Policy confusion
- Employee relations risk

## What the Dashboard Can Help With

- Categorize people problems
- Track diagnostic questions
- Connect issues to KPIs
- Identify repeated patterns
- Flag risk areas
- Recommend next steps
- Track SOP needs
- Document follow-up ownership

## Example Diagnostic Categories

### Broken Trust

Questions to ask:

- Were promises made and not kept?
- Were rules applied differently?
- Was communication delayed or unclear?
- Did employees raise concerns with no follow-up?
- Did leadership explain the decision?

Possible KPIs:

- Follow-up completion rate
- Repeat concern rate
- Engagement score movement
- Manager communication completion
- Employee relations case aging

### Hiring Delays

Questions to ask:

- Where is the process slowing down?
- Who owns each step?
- Are candidates waiting too long?
- Are managers aligned on requirements?
- Are decisions documented?

Possible KPIs:

- Time to fill
- Time to schedule
- Interview feedback completion rate
- Offer acceptance rate
- Candidate response time

### Manager Issues

Questions to ask:

- Are complaints repeating under the same manager?
- Are expectations clear?
- Is documentation consistent?
- Has the manager received support?
- Are employees escalating concerns late?

Possible KPIs:

- ER cases by manager
- Turnover under manager
- Engagement score by manager
- Repeat complaint rate
- Time to escalation

## Sample Dashboard Fields

- Issue category
- Team or department
- Manager
- Risk level
- Related KPIs
- Diagnostic questions
- Follow-up owner
- Status
- SOP needed
- Next step

## Safety Position

The dashboard is a decision-support tool.

It should help leaders see patterns, ask better questions, and improve follow-up.

It should not replace HR judgment, legal review, or direct human conversations.
